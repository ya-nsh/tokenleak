// features

Everything you need to understand your usage

Tokenleak computes deep stats from your local log files — no telemetry, no cloud, no API keys.

🔥

GitHub-style Heatmap

A full-year heatmap using Unicode block chars — see your coding streaks, peak days, and gaps at a glance.

📊

Deep Stats Panel

Current streak, longest streak, 30-day rolling window, total cost, cache hit rate, daily averages — all live from logs.

⚡

Compare Mode

Split your range in half with --compare auto or pick two custom ranges to see exactly how usage changed.

🧠

Model Breakdown

Top N models ranked by token consumption with visual bar charts. Spot when you drifted from Haiku to Opus.

💰

Cost Tracking

Estimated USD cost per day, per model, and rolling projections. Know your monthly burn rate before the invoice.

📦

Cache Economics

Cache reads, writes, read coverage %, and reuse ratio. Understand how well prompt caching is working for you.

📅

Day-of-Week Analysis

Which days of the week do you code most? See your Monday vs Friday AI usage patterns.

🕐

Hour-of-Day Breakdown

A bar chart of your peak coding hours across the day — are you a morning or late-night developer?

🔗

Sharing & Export

--clipboard, --open, and --upload gist to share your stats anywhere.

// providers

All your AI coding tools in one place

Tokenleak auto-detects which tools you have installed. All data is read locally — nothing is sent anywhere.

🟠

Claude Code

~/.claude/projects/*/*.jsonl

Reads JSONL conversation logs. Parses every assistant message with a usage field for input, output, and cache token counts.

🟢

Codex

~/.codex/sessions/YYYY/MM/DD/

Reads JSONL session logs. Parses response events for token usage with cumulative delta extraction.

🟣

Open Code

~/.local/share/opencode/storage/

Reads from the SQLite database or legacy JSON session files, with automatic fallback between storage formats.

// output formats

Terminal. SVG. PNG. JSON.

Four output formats for every use case — from live terminal dashboards to shareable year-in-review cards.

--format terminal

Terminal Dashboard

Full ANSI dashboard in your terminal. Adapts to width — falls back to a compact one-liner on narrow terminals.

  • GitHub-style heatmap with Unicode block chars (░▒▓█)
  • Stats panel, day-of-week chart, hour-of-day bars
  • Top models ranked by usage
  • --no-color mode for piping
--format svg

SVG Heatmap

Self-contained SVG with heatmap grid, stats panel, insights, day-of-week chart, and model breakdown. Dark + light themes.

  • Quantile-based colour intensity
  • Month and day-of-week labels
  • Embeds in READMEs and docs
  • Badge variant for streak counts
--format png

PNG Image

Same layout as SVG rendered to a sharp PNG via the sharp library. 1200×630 Wrapped card for year-in-review sharing.

  • Perfect for social media sharing
  • Shareable year-in-review card
  • --upload gist to GitHub Gist
--format json

Structured JSON

Schema-versioned JSON with daily usage, model breakdowns, and all aggregated stats. Pipe into your own tools.

  • Daily usage per provider
  • Model-level token + cost breakdown
  • All aggregated stats (streaks, rolling, peaks)
  • Compare mode outputs deltas

// install

Get started in seconds

Requires Bun v1.0+. One global install, then just run tokenleak.

  1. 1

    Install Bun (if needed)

    curl -fsSL https://bun.sh/install | bash

  2. 2

    Install Tokenleak globally

    bun install -g tokenleak

  3. 3

    Run in your terminal

    Just tokenleak. It auto-detects providers and opens the interactive launcher.

quick examples 
# Interactive launcher (TTY)
tokenleak

# Terminal dashboard, last 30 days
tokenleak --days 30

# Export SVG heatmap
tokenleak -o usage.svg

# Only Claude Code, light theme
tokenleak --provider claude-code --theme light

# Compare last 60 days vs. prior 60
tokenleak --compare auto --days 120

# Copy JSON to clipboard
tokenleak --format json --clipboard

# Upload PNG to GitHub Gist
tokenleak -o card.png --upload gist
// flags

All flags

Every flag is also available via environment variable (TOKENLEAK_FORMAT, etc.) or ~/.tokenleakrc. CLI flags always win.

Flag Alias Default Description
--format-fterminalOutput format: json, svg, png, terminal
--theme-tdarkColour theme: dark, light
--days-d90Number of days to look back
--since-s—Start date (YYYY-MM-DD). Overrides --days
--until-utodayEnd date (YYYY-MM-DD)
--output-ostdoutOutput file path. Format is inferred from extension
--provider-pallFilter to specific provider(s), comma-separated
--compare——Compare two ranges: auto or YYYY-MM-DD..YYYY-MM-DD
--width-w80Terminal width for dashboard layout
--no-color—falseStrip ANSI escape codes from terminal output
--no-insights—falseHide the insights panel
--clipboard—falseCopy output to clipboard after rendering
--open—falseOpen output file in default app (requires --output)
--upload——Upload to a service. Supported: gist
--version——Print version number
--help——Print usage information
